## Promesse client

MadaVoyage aide les voyageurs a choisir un circuit adapte, comprendre les
destinations, verifier les departs disponibles et envoyer une demande de
reservation. L'equipe confirme ensuite manuellement les disponibilites, les
details du voyage et les prochaines etapes.

La reservation faite sur le site ou par l'agent vocal est une demande de
reservation. Ce n'est pas un paiement final et ce n'est pas une confirmation
definitive avant validation humaine par l'equipe MadaVoyage.

## Destinations

### Antananarivo

Region : Hautes terres centrales.

Resume : capitale en hauteur melant palais royaux, collines historiques,
marches vivants et panoramas urbains.

Introduction : Antananarivo est souvent la premiere rencontre avec Madagascar.
La ville se decouvre par ses hauteurs, ses quartiers anciens, ses marches et
ses sites royaux autour de la capitale.

Points forts : Rova d'Antananarivo, Haute-Ville, Analakely, Lac Anosy.

Ideal pour : culture, histoire, premiere immersion.

Duree conseillee : 1 a 2 jours.

### Tsingy de Bemaraha

Region : Ouest de Madagascar.

Resume : paysage mineral spectaculaire fait d'aiguilles calcaires, de
passerelles et de canyons naturels.

Introduction : les Tsingy de Bemaraha offrent l'un des decors les plus
impressionnants de Madagascar, entre formations calcaires acerees, ponts
suspendus et forets seches.

Points forts : Grands Tsingy, Petits Tsingy, Gorges de Manambolo, foret seche.

Ideal pour : aventure, paysages rares, photographie.

Duree conseillee : 2 a 3 jours.

### Allee des baobabs

Region : Morondava.

Resume : image iconique de Madagascar, avec ses baobabs geants alignes dans la
lumiere de fin de jour.

Introduction : l'Allee des baobabs est une etape incontournable pres de
Morondava. Le lieu est particulierement fort au lever ou au coucher du soleil.

Points forts : baobabs geants, coucher de soleil, Morondava, villages de
l'ouest.

Ideal pour : photo, coucher de soleil, voyage iconique.

Duree conseillee : 1 jour.

### Plages paradisiaques

Region : cotes et iles.

Resume : lagons, sable clair et rythme insulaire pour terminer un circuit par
une vraie respiration.

Introduction : les plages de Madagascar permettent de combiner aventure et
repos. Selon la saison, l'itineraire peut viser Nosy Be, Sainte-Marie ou
d'autres cotes plus confidentielles.

Points forts : Nosy Be, Sainte-Marie, lagons, snorkeling.

Ideal pour : repos, mer, fin de circuit.

Duree conseillee : 3 a 5 jours.

### Lemuriens et forets

Region : parcs et reserves.

Resume : immersion dans les forets malgaches pour observer lemuriens,
cameleons et biodiversite endemique.

Introduction : les lemuriens sont au coeur de l'identite naturelle de
Madagascar. Les parcs et reserves permettent de les observer avec un guide
local, dans le respect des lieux.

Points forts : Andasibe, Ranomafana, guides naturalistes, biodiversite.

Ideal pour : nature, faune, familles curieuses.

Duree conseillee : 2 a 4 jours.

## FAQ

Question : Quelle est la meilleure periode pour voyager a Madagascar ?

Reponse : La saison seche, de mai a octobre, est souvent la plus confortable
pour les circuits. Certaines regions restent interessantes toute l'annee selon
l'itineraire choisi.

Question : Les circuits sont-ils accompagnes par un guide local ?

Reponse : Oui, les circuits proposes sont concus avec un accompagnement local
pour faciliter les trajets, les visites et la comprehension des lieux
traverses.

Question : Peut-on demander une date personnalisee ?

Reponse : Oui, le formulaire permet de demander une date sur mesure, un depart
prive ou une adaptation du circuit selon le nombre de voyageurs.

Question : Le site peut-il etre adapte pour un autre client ?

Reponse : Oui, l'architecture separe les donnees, les sections et la
configuration de marque pour permettre une duplication rapide vers une autre
agence ou un autre metier.

## Regles de reservation

L'agent peut aider le voyageur a choisir un circuit, expliquer les departs et
preparer une demande de reservation.

Avant de creer une demande, l'agent doit obtenir :

- le circuit souhaite ;
- le type de depart : depart planifie ou date personnalisee ;
- la date ou le depart choisi ;
- le nombre de voyageurs ;
- le nom du voyageur ;
- l'adresse email ;
- le telephone si possible ;
- un message optionnel si le besoin est specifique.

Pour un depart planifie, l'agent doit appeler l'outil de disponibilite avant de
confirmer qu'il reste assez de places.

La demande de reservation n'est pas une confirmation finale. L'equipe
MadaVoyage verifie les disponibilites, les conditions et revient vers le
voyageur.

Ne jamais promettre un paiement, un visa, une garantie administrative, un
diagnostic medical ou une disponibilite non verifiee.

## Escalade humaine

L'agent doit proposer de transmettre a l'equipe si :

- le voyageur demande un prix special ou une negotiation ;
- le voyageur parle d'une contrainte medicale, administrative ou familiale
  importante ;
- le voyageur veut un itineraire tres personnalise ;
- le voyageur est mecontent ou inquiet ;
- l'agent n'est pas sur de la reponse ;
- la disponibilite ne peut pas etre verifiee ;
- le sujet sort du cadre MadaVoyage.
